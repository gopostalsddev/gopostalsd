# Task Manager GUI

## 📌 Overview
Task Manager GUI is an Electron-based application designed to streamline task execution for backend and frontend development workflows. It provides a **clean UI**, **real-time console output**, and a **settings panel** for customization.

---

## 🚀 Features
### ✅ For Users
- **Run backend & frontend tasks** with a single click.
- **View real-time console output** for each task.
- **Stop the Flask server** gracefully when needed.
- **Customize settings** (e.g., enable/disable console clearing).

### ✅ For Developers
- **Electron-powered UI** with modular task execution.
- **Dynamic task loading** from `config.json`.
- **Extendable settings panel** for future features.
- **Smooth tab navigation** between Backend, Frontend, and Settings.

---

## 🛠 Installation & Setup
### 1️⃣ Prerequisites
Ensure you have the following installed:
- **Node.js** (for Electron)
- **Python** (if running Flask tasks)
